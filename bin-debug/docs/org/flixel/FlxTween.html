Packageorg.flixel
Classpublic class FlxTween
InheritanceFlxTween Object

Simple class for tweening a simple numerical value from one point to another. For more complex operations, please use a dedicated library such as TweenMax. This class does not use the "global" time to progress the tween, but must manually be incremented with the `progress` property.
